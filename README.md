# WAL-E (Walmart Advertising Learning Engine)

## Vision
WAL-E is an enterprise-grade AI knowledge platform focused on Walmart Connect, Display Self-Serve (DSS), Onsite Display, and Advertiser Onboarding Specialists (AOS).

## Objectives
- Build the most complete public AI-ready knowledge base.
- Train AI agents using structured documentation.
- Support RAG pipelines.
- Provide troubleshooting, SOPs, FAQs, business rules, and learning paths.
