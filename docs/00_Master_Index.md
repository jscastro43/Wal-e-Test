# Walmart Connect Enterprise AI Knowledge Base

## Volume 00 - Master Index

This repository is the foundation of the WAL-E (Walmart Advertising Learning Engine) project.

## Mission
Build a comprehensive AI-ready knowledge base for Walmart Connect Display Self-Service (DSS), Onsite Display, and Advertiser Onboarding Specialists (AOS).

## Repository Structure
- docs/
- knowledge/
- datasets/
- prompts/
- src/
- web/

## Documentation Standard
Every document will include:
1. Metadata
2. Overview
3. Definitions
4. Business Context
5. Workflow
6. Business Rules
7. Troubleshooting
8. FAQs
9. References
10. AI Training Notes
